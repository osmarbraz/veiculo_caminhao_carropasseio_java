# Programa Veiculo, Caminhão e Carro de Passeio.

1 Escreva a classe Veiculo contendo Peso em quilos(int), VelocMax em Km/h(int) e Preco em R$ (float).

a) Crie um construtor que inicialize os dados com zero e “”.

b) Crie um construtor que inicialize os dados com valores passados por parâmetro.

c) Crie métodos set’s para alterar os dados.

d) Crie métodos get’s para retornar os dados.

e) Crie um método  para solicitar os dados para o usuário.

f) Crie um método para imprimir os dados  para o usuário.

g) Construa um programa para testar a classe Veiculo.
